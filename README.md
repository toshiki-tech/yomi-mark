# YomiMark — Instant Furigana for the Web

YomiMark 是一个简洁高效的 Chrome 浏览器扩展，致力于帮助日语学习者和阅读者快速掌握汉字读音。

## 核心功能

- **划词注音**：在网页上选中一段包含汉字的日语文本，点击出现的「読」浮动按钮，即可瞬间在汉字上方添加假名注音（furigana）。
- **词级智能分词**：采用高效的形態素解析器（kuromoji.js），确保注音准确且符合词汇边界。
- **无侵入设计**：仅处理用户选中的内容，不破坏原始网页结构或脚本逻辑。
- **离线工作**：内置字典文件，无需将数据发送到远程服务器，保护隐私且响应极快。

## 安装指南 (开发者模式)

由于本项目目前处于开发阶段，您可以按照以下步骤手动加载到浏览器：

1. 下载或克隆本项目到您的本地目录。
2. 打开 Google Chrome 浏览器，访问 `chrome://extensions/`。
3. 在右上角开启 **"开发者模式" (Developer mode)**。
4. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5. 选择项目根目录（包含 `manifest.json` 的文件夹）。
6. YomiMark 图标将出现在您的扩展栏中。

## 使用方法

1. 浏览任意含日语的网页。
2. 选中您想要查看读音的句子或短语。
3. 点击选区上方出现的紫色 **「読」** 按钮。
4. 您会看到汉字上方精准刻上了注音。

## 技术架构

- **Manifest V3**: 符合 Chrome 扩展最新标准。
- **kuromoji.js**: 在客户端运行的形态素解析。
- **HTML5 `<ruby>`**: 使用原生 Web 标准实现注音渲染。
