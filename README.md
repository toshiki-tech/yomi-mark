# YomiMark — Instant Furigana for the Web

YomiMark 是一个极简、高效且隐私优先的 Chrome 浏览器扩展，致力于帮助日语学习者和阅读者快速掌握汉字读音，并解锁神秘的“外来语”含义。

## ✨ 核心特性

- **🎯 划词触发**：选中一段日语文本，点击出现的「読」浮动按钮，即可瞬间在汉字上方添加假名注音（furigana）。拒绝全选注音造成的网页排版混乱。
- **📖 外来语翻译 (Gairaigo)**：特别集成外来语大词典。当选中片假名单词（如「コンテキスト」）时，插件会自动显示其英语原词（Context），让外来语不再是阅读障碍。
- **🛡️ 隐私优先 & 零延迟**：内置完整的形态素解析引擎（kuromoji.js）和全量离线字典。**所有处理均在本地进行**，不向任何服务器发送您的阅读内容，隐私无忧且响应极快。
- **🎨 高度可定制**：通过扩展弹出窗口（Popup），您可以自由调整汉字注音和外来语翻译的颜色，完美适配您的阅读偏好。
- **💎 无侵入设计**：仅处理用户选中的内容，使用原生 HTML `<ruby>` 标签渲染，保证排版整洁、文字可复制。
- **词级智能分词**：采用改进的形态素解析器，支持复合词最长匹配逻辑，确保注音准确且符合词汇边界。

## 📥 安装指南 (开发者模式)

由于本项目目前处于开发阶段，您可以按照以下步骤手动加载到浏览器：

1. 下载或克隆本项目到您的本地目录。
2. 打开 Google Chrome 浏览器，访问 `chrome://extensions/`。
3. 在右上角开启 **"开发者模式" (Developer mode)**。
4. 点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5. 选择项目根目录 `c:\toshiki-tech-lab\yomi-mark`。
6. YomiMark 图标将出现在您的扩展栏中。

## 🖱️ 使用方法

1. 浏览任意含日语的网页。
2. 选中您想要查看读音或含义的句子、短语或单词。
3. 点击选区上方出现的紫色 **「読」** 按钮。
4. 您会看到汉字上方精准刻上了注音，外来语上方出现了英语原词。
5. 点击插件图标，在弹出窗口中可自由调整注音颜色。

## 🏗️ 技术架构

- **Manifest V3**: 符合 Chrome 扩展最新标准，长效稳定。
- **kuromoji.js**: 在客户端运行的高性能形态素解析。
- **Offline Dict**: 预加载的分级词典，支持汉字读音与外来语映射。
- **HTML5 `<ruby>`**: 使用 Web 标准实现注音渲染，保证最佳排版。

## 💡 为什么选择 YomiMark？

传统的日语注音扩展往往会瞬间“污染”整个网页，导致排版错位、CPU 占用飙升。YomiMark 坚持**“只在需要时出现”**的原则，它不仅是一个工具，更是您在日语阅读海洋中随手可得的“放大镜”。
